Javascript - html을 동적으로 제어할 수 있는 스크립트 언어
JQuery - Javascript를 간소화시켜놓은 라이브러리
Node.js - Javascript로 서버측 기술을 구현해놓은 것

개발자들은 완성된 프로그램 소스를 가지고 있지 않는다.
-> 내부 IP라 외부의 사용자가 들어올 수 없다 (유일하지 않은 IP를 할당받았기 때문에)
-> 공인 IP의 회사의 서버 컴퓨터에 등록해놔야함

클라이언트 사이드 - 서버 사이드 

Node.js란?
- 자바스크립트다.
- 단, HTML을 제어하기 위함이 아니라, 서버측의 기술을 구현하기 위한 
자바스크립트이다.
- 즉, 웹서버 구축이나 채팅서버 구축에 사용됨
- DOM은 없음 (html을 제어하지 않아서)
- 서버구축에 필요한 기술(모듈, 전역객체 ...)가 대체
